Descrição do sistema: É um cadastro de um evento de luta 

Decisões técnicas:Utilizei três vetores paralelos, (vetorNome, vetorCartel, vetorData, vetorStatus,) 
para armazenar as informções dos lutadores.
O sistema utiliza um laço (while) para manter a execução do menu.
A lógica de busca e listagem percorre os vetores com um (for) simples, comparando entradas com o indice (totalRegistros) para limitar a execução para os dados cadastrados.
Eu não soube usar swift e case então utilizei if,else
